Discord Bot Project
===================

이 프로젝트는 Discord 봇을 위한 기본적인 구조를 제공합니다. 봇은 다양한 기능을 모듈화하여 관리할 수 있도록 설계되었습니다.

구조
-----
- `bot.py`: 봇 실행 및 설정 파일입니다.
- `cogs/`: 각 기능을 모듈화한 폴더입니다.
  - `__init__.py`: cogs 폴더를 패키지로 인식하게 합니다.
  - `general.py`: 일반 명령어를 처리하는 파일입니다.
  - `game.py`: 게임 관련 명령어를 처리하는 파일입니다.
  - `utility.py`: 유틸리티 명령어를 처리하는 파일입니다.
  - `weather.py`: 날씨 관련 명령어를 처리하는 파일입니다.
- `requirements.txt`: 필요한 라이브러리 목록입니다.
- `config.py`: 봇 설정 파일입니다.
- `data/`: 데이터 저장 폴더입니다.

설치 및 실행
-------------
1. 필요한 라이브러리 설치:
   ```
   pip install -r requirements.txt
   ```

2. `config.py` 파일을 열어 `DISCORD_TOKEN`과 `API_KEY`를 실제 값으로 설정합니다.

3. 봇 실행:
   ```
   python bot.py
   ```

기능
-----
- `!hello`: 일반 명령어로, 봇이 "Hello!"라고 응답합니다.
- `!roll`: 게임 명령어로, 주사위를 굴리는 메시지를 보냅니다.
- `!ping`: 유틸리티 명령어로, 봇이 "Pong!"이라고 응답합니다.
- `!weather <location>`: 날씨 명령어로, 지정된 위치의 날씨 정보를 가져옵니다.

이 프로젝트는 Discord 봇 개발을 시작하는 데 유용한 기본 틀을 제공합니다. 각 기능은 `cogs` 폴더 내의 모듈로 쉽게 확장할 수 있습니다.
